## Imagens

* <CheckableBox/> Adicionar `alt` para imagens, botões-imagem, gráficos e imagens de mapas com pontos de acesso.
* <CheckableBox/> Adicionar `alt=""` para imagens decorativas que não tem significado.
* <CheckableBox/> Incluir a descrição no `alt` para imagens que contém texto.
* <CheckableBox/> Garantir contraste adequado entre texto da imagem e plano de fundo.
* <CheckableBox/> Evitar textos com fontes muito pequenas dentro da imagem.

## Vídeos

* <CheckableBox/> Adicionar legendas para áudio em vídeos existentes.
* <CheckableBox/> Evitar conteúdo com flashes (gatilhos para convulsões) ou manter abaixo dos limites.
* <CheckableBox/> Adicionar botão de controle de legenda e audiodescrição.

## Controles

* <CheckableBox/> Adicionar `href` para links.
* <CheckableBox/> Adicionar underline nos links.
* <CheckableBox/> Adicionar estados de foco em campos de entrada, botões, e elementos interativos.
* <CheckableBox/> Adicionar `type="button"` nos botões.
* <CheckableBox/> Adicionar skip-link (link para pular) para o conteúdo principal.
* <CheckableBox/> Identifique e comunique links que abrem em uma nova guia ou janela.
* <CheckableBox/> Garantir que todos os botões tenham nome acessível por leitores de tela.

## Formulário

* <CheckableBox/> Adicionar `label` para os campos de entradas associadas ao elemento correspondente.
* <CheckableBox/> Adicionar `<fildset>` e `<legend>` para seção no formulário.
* <CheckableBox/> Adicionar `autocomplete` para campos de entrada.
* <CheckableBox/> Exibir errors (erros) de entrada acima do formulário, após envio.
* <CheckableBox/> Adicionar `aria-describedby` para os campos de entrada.
* <CheckableBox/> Exibir mensagens de erro e sucesso não só visualmente.
* <CheckableBox/> Utilizar validação em tempo real acessível para indicar campos obrigatórios ou incorretos.

## Mídia

* <CheckableBox/> Impedir `autoplay` para vídeos e audios.
* <CheckableBox/> Adicionar `type` para botões e entradas.
* <CheckableBox/> Adicionar pausa para todas as mídias.
* <CheckableBox/> Adicionar transcrição para audios.

## Semântica

* <CheckableBox/> Uso de elementos nativos HTML.
* <CheckableBox/> Fluxo continuo e Lógico.
* <CheckableBox/> Tem descrições que podem ser facilmente compreendidas.
* <CheckableBox/> Tem a semântica correta.
* <CheckableBox/> É objetivo nos rótulos.

## Texto

* <CheckableBox/> Evitar o uso de textos dentro de imagens.
* <CheckableBox/> Redimensiona os textos na página, aumentando o zoom em até 200%.
* <CheckableBox/> Alturas das fontes não é fixa.

## Teclado

* <CheckableBox/> Funcionalidades da página web estão disponíveis por teclado.
* <CheckableBox/> Quando se tem o mouseover é permitido o uso de teclado.
* <CheckableBox/> Foco visível remova elementos focalizáveis invisíveis.
* <CheckableBox/> Adicione o `.hover`, `.focus` para tornar o foco visível.
* <CheckableBox/> Permite/visa o uso de Atalhos de teclado como o TAB.
* <CheckableBox/> Primeiro item interativo da página é um link para o conteúdo principal.
* <CheckableBox/> Garantir ordem lógica de navegação por teclado.

## Título

* <CheckableBox/> A hierarquia de conteúdo da página é definida por sua lógica não pelo tamanaho do texto.
* <CheckableBox/> Use elementos de título `h1` `h2` `h3` para apresentar o conteúdo.
* <CheckableBox/> Não pular níveis lógicos.
* <CheckableBox/> Toda página contem um título `h1` descrevendo a página.
* <CheckableBox/> Tabela
* <CheckableBox/> Use o `table` para elementos em formato de tabela.
* <CheckableBox/> Insira cabeçalhos para explicar os dados, use `th` com `scope` correto.
* <CheckableBox/> Use o `caption` elemento para fornecer um título para a tabela.
* <CheckableBox/> Não usar tabelas para layout.

## Modais

* <CheckableBox/> Deve ser fácil fechar.
* <CheckableBox/> Permiti o uso da tecla escape ESC.
* <CheckableBox/> A interação é uma tarefa simples.
* <CheckableBox/> Evita modais em tela cheia.
* <CheckableBox/> Não abrir um modal a partir de outro modal.
* <CheckableBox/> Gerenciar foco corretamente dentro do modal.

## Dispositivo Móvel e tocável

* <CheckableBox/> O site pode ser rotacionado para qualquer orientação.
* <CheckableBox/> Impedir rolgem horizontal.
* <CheckableBox/> Garantir que botões e links possam ser ativados facilmente.
* <CheckableBox/> Garantir espaço suficiente entre elementos interativos.
* <CheckableBox/> Garantir acessibilidade para gestos de toque.

## Ferramentas e extras

* <CheckableBox/> Permiti pausar, parar ou ocultar conteúdo em movimento.
* <CheckableBox/> Usar Breadcrumbs informando a localização atual nas páginas.
* <CheckableBox/> Colocar página ou área de esclarecimento de dúvidas e dicas de acessibilidade.
* <CheckableBox/> Áreas clicáveis com no mínimo 44px (pixels) de altura e 44px de largura.
* <CheckableBox/> No caso de captcha garanta que seja simples de entender e tenha alternativas para pessoas com deficiência.
* <CheckableBox/> Incluir um campo de busca.
* <CheckableBox/> Indicar idioma principal da página com `lang=""` na tag `<html>`.
