# Desenvolvimento Web

## Imagens

* <CheckableBox id='dev-img-1' /> Adicionar `alt` para imagens, botões-imagem, gráficos e imagens de mapas com pontos de acesso.
* <CheckableBox id='dev-img-2' /> Adicionar `alt=""` para imagens decorativas que não tem significado.
* <CheckableBox id='dev-img-3' /> Incluir a descrição no `alt` para imagens que contém texto.
* <CheckableBox id='dev-img-4'/> Garantir contraste adequado entre texto da imagem e plano de fundo.
* <CheckableBox id='dev-img-5'/> Evitar textos com fontes muito pequenas dentro da imagem.

## Vídeos

* <CheckableBox id='dev-vid-1'/> Adicionar legendas para áudio em vídeos existentes.
* <CheckableBox id='dev-vid-2'/> Evitar conteúdo com flashes (gatilhos para convulsões) ou manter abaixo dos limites.
* <CheckableBox id='dev-vid-3'/> Adicionar botão de controle de legenda e audiodescrição.

## Controles

* <CheckableBox id='dev-ctrl-1'/> Adicionar `href` para links.
* <CheckableBox id='dev-ctrl-2'/> Adicionar underline nos links.
* <CheckableBox id='dev-ctrl-3'/> Adicionar estados de foco em campos de entrada, botões, e elementos interativos.
* <CheckableBox id='dev-ctrl-4'/> Adicionar `type="button"` nos botões.
* <CheckableBox id='dev-ctrl-5'/> Adicionar skip-link (link para pular) para o conteúdo principal.
* <CheckableBox id='dev-ctrl-6'/> Identifique e comunique links que abrem em uma nova guia ou janela.
* <CheckableBox id='dev-ctrl-7'/> Garantir que todos os botões tenham nome acessível por leitores de tela.

## Formulário

* <CheckableBox id='dev-form-1'/> Adicionar `label` para os campos de entradas associadas ao elemento correspondente.
* <CheckableBox id='dev-form-2'/> Adicionar `<fildset>` e `<legend>` para seção no formulário.
* <CheckableBox id='dev-form-3'/> Adicionar `autocomplete` para campos de entrada.
* <CheckableBox id='dev-form-4'/> Exibir errors (erros) de entrada acima do formulário, após envio.
* <CheckableBox id='dev-form-5'/> Adicionar `aria-describedby` para os campos de entrada.
* <CheckableBox id='dev-form-6'/> Exibir mensagens de erro e sucesso não só visualmente.
* <CheckableBox id='dev-form-7'/> Utilizar validação em tempo real acessível para indicar campos obrigatórios ou incorretos.

## Mídia

* <CheckableBox id='dev-media-1'/> Impedir `autoplay` para vídeos e audios.
* <CheckableBox id='dev-media-2'/> Adicionar `type` para botões e entradas.
* <CheckableBox id='dev-media-3'/> Adicionar pausa para todas as mídias.
* <CheckableBox id='dev-media-4'/> Adicionar transcrição para audios.

## Semântica

* <CheckableBox id='dev-sem-1'/> Uso de elementos nativos HTML.
* <CheckableBox id='dev-sem-2'/> Fluxo continuo e Lógico.
* <CheckableBox id='dev-sem-3'/> Tem descrições que podem ser facilmente compreendidas.
* <CheckableBox id='dev-sem-4'/> Tem a semântica correta.
* <CheckableBox id='dev-sem-5'/> É objetivo nos rótulos.

## Texto

* <CheckableBox id='dev-txt-1'/> Evitar o uso de textos dentro de imagens.
* <CheckableBox id='dev-txt-2'/> Redimensiona os textos na página, aumentando o zoom em até 200%.
* <CheckableBox id='dev-txt-3'/> Alturas das fontes não é fixa.

## Teclado

* <CheckableBox id='dev-key-1'/> Funcionalidades da página web estão disponíveis por teclado.
* <CheckableBox id='dev-key-2'/> Quando se tem o mouseover é permitido o uso de teclado.
* <CheckableBox id='dev-key-3'/> Foco visível remova elementos focalizáveis invisíveis.
* <CheckableBox id='dev-key-4'/> Adicione o `.hover`, `.focus` para tornar o foco visível.
* <CheckableBox id='dev-key-5'/> Permite/visa o uso de Atalhos de teclado como o TAB.
* <CheckableBox id='dev-key-6'/> Primeiro item interativo da página é um link para o conteúdo principal.
* <CheckableBox id='dev-key-7'/> Garantir ordem lógica de navegação por teclado.

## Título

* <CheckableBox id='dev-title-1'/> A hierarquia de conteúdo da página é definida por sua lógica não pelo tamanaho do texto.
* <CheckableBox id='dev-title-2'/> Use elementos de título `h1` `h2` `h3` para apresentar o conteúdo.
* <CheckableBox id='dev-title-3'/> Não pular níveis lógicos.
* <CheckableBox id='dev-title-4'/> Toda página contem um título `h1` descrevendo a página.

## Tabela

* <CheckableBox id='dev-tbl-1'/> Use o `table` para elementos em formato de tabela.
* <CheckableBox id='dev-tbl-2'/> Insira cabeçalhos para explicar os dados, use `th` com `scope` correto.
* <CheckableBox id='dev-tbl-3'/> Use o `caption` elemento para fornecer um título para a tabela.
* <CheckableBox id='dev-tbl-4'/> Não usar tabelas para layout.

## Modais

* <CheckableBox id='dev-modal-1'/> Deve ser fácil fechar.
* <CheckableBox id='dev-modal-2'/> Permiti o uso da tecla escape ESC.
* <CheckableBox id='dev-modal-3'/> A interação é uma tarefa simples.
* <CheckableBox id='dev-modal-4'/> Evita modais em tela cheia.
* <CheckableBox id='dev-modal-5'/> Não abrir um modal a partir de outro modal.
* <CheckableBox id='dev-modal-6'/> Gerenciar foco corretamente dentro do modal.

## Dispositivo Móvel e tocável

* <CheckableBox id='dev-mob-1'/> O site pode ser rotacionado para qualquer orientação.
* <CheckableBox id='dev-mob-2'/> Impedir rolgem horizontal.
* <CheckableBox id='dev-mob-3'/> Garantir que botões e links possam ser ativados facilmente.
* <CheckableBox id='dev-mob-4'/> Garantir espaço suficiente entre elementos interativos.
* <CheckableBox id='dev-mob-5'/> Garantir acessibilidade para gestos de toque.

## Ferramentas e extras

* <CheckableBox id='dev-extra-1'/> Permiti pausar, parar ou ocultar conteúdo em movimento.
* <CheckableBox id='dev-extra-2'/> Usar Breadcrumbs informando a localização atual nas páginas.
* <CheckableBox id='dev-extra-3'/> Colocar página ou área de esclarecimento de dúvidas e dicas de acessibilidade.
* <CheckableBox id='dev-extra-4'/> Áreas clicáveis com no mínimo 44px (pixels) de altura e 44px de largura.
* <CheckableBox id='dev-extra-5'/> No caso de captcha garanta que seja simples de entender e tenha alternativas para pessoas com deficiência.
* <CheckableBox id='dev-extra-6'/> Incluir um campo de busca.
* <CheckableBox id='dev-extra-7'/> Indicar idioma principal da página com `lang=""` na tag `<html>`.
